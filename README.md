Healthcare-Analytics-Snowflake
End-to-End Healthcare Analytics Project using Snowflake and Power BI

The project showcases:

- Data Ingestion
- RAW / STAGING / CURATED Architecture
- Dimensional Modeling
- Fact & Dimension Tables
- Reporting Views
- Interactive Dashboard Development

## Technology Stack

- Snowflake
- SQL
- Power BI
- DAX
- Data Modeling
- Star Schema
